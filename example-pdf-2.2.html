ComponentViewer — PDF.js 2.2.228

This page uses PDF.js 2.2.228 to verify the plugin works with the 2.2.x API. PDF rendering, thumbnails, zoom, rotate, and annotations (links, highlights) are supported with an internal fallback when pdfjsLib.Util is not exposed.

example.html uses PDF.js 3.x. example-no-libs.html uses no PDF.js (iframe fallback).

PDF with PDF.js 2.2.228

Click the PDF or image to open the viewer. The plugin uses pdf.workerSrc for the 2.2.228 worker. Page navigation, thumbnails, zoom, rotate, print, and annotations (if the PDF has links/highlights) work with this version.
