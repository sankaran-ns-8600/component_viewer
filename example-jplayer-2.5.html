ComponentViewer — jPlayer 2.5.0

This page uses jPlayer 2.5.0 for video and audio playback. The plugin’s built-in video and audio renderers use jPlayer’s API (setMedia, play, pause, destroy, cssSelectorAncestor, etc.), which is compatible with 2.5.0.

example.html uses jPlayer 2.9.2. example-no-libs.html uses no jPlayer (native HTML5 fallback).
